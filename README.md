Final_project

This project is a basic implementation of the Tetris game developed using C++ as part of a semester final project. The objective was to understand object-oriented programming concepts, game logic, and code structuring using header and source files.

## Technologies Used
- C++
- QMake

## Features
- Basic Tetris gameplay logic
- Grid and block management
- Modular code structure using multiple classes

## Learning Outcomes
- Improved understanding of C++ classes and header files
- Experience with structuring a medium-sized project
- Logical thinking and debugging skills
